# Homebrew Tap for OpenPencil

Official Homebrew tap for installing OpenPencil on macOS.

This repository contains the Homebrew cask used to distribute OpenPencil, the open-source AI-native vector design tool from ZSeven-W.

## Install

```bash
brew tap zseven-w/openpencil
brew install --cask openpencil
```

## Upgrade

```bash
brew upgrade --cask openpencil
```

## Uninstall

```bash
brew uninstall --cask openpencil
```

## Verify

After installation, launch OpenPencil from Applications or with:

```bash
open -a OpenPencil
```

## About OpenPencil

OpenPencil is a design-as-code canvas for turning prompts into live UI. Learn more here:

- Main repo: https://github.com/ZSeven-W/openpencil
- Website: https://op.zseven.tech
